稍微進階是對我來說啦

grep

find 找檔案

locate 找檔案(查系統index，不是直接掃目錄)

  • Ctrl + c : 終止正在執行的程式

  • Ctrl + l : clear

  • Ctrl + w : 刪除文字至上一個空白

  • Ctrl + u : 刪除游標前
  • Ctrl + k : 刪除游標後
  • Ctrl + h : Backspace

  • Ctrl + d : Del

  • Ctrl + a : 游標移到最前面

  • Ctrl + e : 游標移到最後面
  • Ctrl + f : 往右移一個字元
  • Ctrl + b : 往左移一個字元
