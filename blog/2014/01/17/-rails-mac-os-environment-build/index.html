[Mac] Ruby on Rails 基本環境建置

是說終於買了mac，省去開箱文趕緊把環境用好來繼續練習ROR!! 但使用上還是有一點不習慣，會覺得有些地方也蠻不人性化，但相對穩定就是了

以下環境為OS X 10.9.1

1.安裝xcode-select Command line tool已經獨立出來，不需要安裝XCode了

如果安裝失敗的話也可以到這裡下載

2.安裝homebrew

3.安裝所需要的工具

4.安裝RVM

5.透過RVM來安裝RUBY

6.透過RVM來安裝RubyGems

7.安裝Rails

XCode:官方提供的一個IDE，主要用來開發IOS APP，不過它也包含了一些開發其他語言所需的環境 Homebrew: OSX 套件管理工具 RVM (Ruby Version Manager):Ruby版本管理工具，便於切換不同版本
